# Clone Frontend: Portale News (My Suburban Life)

Questo progetto è una riproduzione frontend del portale di notizie locale statunitense "My Suburban Life" (rete Shaw Local). Lo scopo del progetto è dimostrare la capacità di replicare interfacce utente complesse e ricche di contenuti, tipiche delle testate giornalistiche.

**[Visualizza il sito live su GitHub Pages](https://frago0312.github.io/ppm-mysuburbanlife/)**

## Caratteristiche del Progetto

* **Layout Completamente Responsive:** Il design si adatta in modo fluido a schermi desktop, tablet e dispositivi mobili, garantendo una leggibilità ottimale in ogni condizione.
* **Sistema a Griglia:** Utilizzo estensivo dei sistemi a griglia per organizzare la complessa struttura a colonne degli articoli, le barre laterali e gli spazi pubblicitari, mantenendo le proporzioni originali.
* **Organizzazione Strutturale:** Strutturazione semantica del codice per separare in modo chiaro la navigazione principale, gli articoli in evidenza e i feed di notizie secondari.

## Stack Tecnologico

* **HTML5:** Per la strutturazione semantica dei contenuti.
* **CSS3:** Per lo styling personalizzato e le media queries.
* **Bootstrap:** Per la gestione rapida del layout responsive e l'implementazione dei componenti dell'interfaccia.
* **JavaScript:** Per la gestione della logica lato client e dell'interattività.
